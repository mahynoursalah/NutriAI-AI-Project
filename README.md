# NutriAI

NutriAI is an AI-powered nutrition and wellness web application designed to help users make smarter dietary decisions through intelligent assistance. The project was developed as an RAG-inspired solution that combines modern web development with OpenAI integration to provide personalized guidance for users interested in improving their health and lifestyle.

The system allows users to track meals, monitor water intake, record weight changes, and receive AI-generated recommendations tailored to their daily habits and wellness goals. By blending nutrition tracking with conversational AI, NutriAI creates a more engaging and practical experience for managing personal health.

## 🚀 Features

- User registration and authentication
- Meal tracking and nutrition monitoring
- Water intake tracking
- Weight tracking and progress monitoring
- AI-powered personalized recommendations
- Smart wellness and diet guidance
- Responsive and modern user interface
- Admin panel for system configuration and management

## 🧠 AI and RAG Concept

NutriAI was built around the idea of combining AI-driven conversational capabilities with structured health and nutrition information. Inspired by RAG (Retrieval-Augmented Generation), the project explores how large language models can be integrated with domain-specific information to provide more relevant and useful responses for users.

This makes the application more than a simple tracking tool—it becomes an intelligent assistant that helps users understand their habits and take better decisions regarding diet and wellness.
